Apresentação Tecnológica

Esta página tem como objetivo destacar as tecnologias e práticas que utilizei no desenvolvimento do projeto. Em baixo está uma lista das tecnologias usadas, juntamente com uma breve descrição de como cada uma delas foi aplicada ao projeto.

  • Django

    Foi a framework web escolhida para este projeto, aproveitando sua modularidade e eficiência em desenvolver aplicações web de forma rápida e eficaz.

  • Python

    Foi a linguagem de programação principal utilizada para desenvolver este projeto Django. Foi usada em diversos ficheiros como os do Models, Views, URLS, Loaders e até Forms... todos estes incluem '.py' no final do seu nome.

  • HTML

    Foi utilizado para estruturar e dar semântica às páginas criadas, garantindo uma base sólida para o conteúdo apresentado e a disponibilização para navegar entre páginas.

  • CSS

    Foi aplicado para estilizar as páginas, garantindo uma apresentação atraente e responsiva para diferentes dispositivos e modos (claro e escuro). O CSS fortificou a diferença entre as aplicações criadas, cada uma com o seu próprio estilo.

  • JavaScript

    Foi utilizado para adicionar interatividade ao site, como a funcionalidade de alternar a visibilidade das seções, trocar o conteúdo da página, adicionar data e hora atulizada ao segundo, entre outros...

  • Git

    Foi utilizado para o controlo de versão, permitindo a identificação de mudanças e a colaboração eficiente no final do desenvolvimento do projeto.

  • GitHub

    Serviu como plataforma de host do código-fonte, facilitando a colaboração e a gestão do projeto, bem como um histórico de versões do projeto.
